## 🌍 Global Trade Intelligence Dashboard

An AI-powered analytics dashboard that evaluates global trade indicators, forecasts export growth, and recommends optimal international markets for expansion.

This project simulates how consulting firms and trade analysts evaluate global markets using economic indicators and data-driven insights.

---

## 🚀 Features

📊 Country Comparison Analytics  
Compare export performance, GDP, inflation, and growth across countries.

📈 Export Performance Forecast  
Analyze historical export data and predict future trade trends.

🏭 Industry Opportunity Forecast  
Identify industries with the highest export potential:
- Electronics
- Machinery
- Chemicals
- Pharmaceuticals
- Clothing
- Grains
- Food Processing

⚖ Decision Intelligence Matrix  
Evaluate trade expansion strategies using Opportunity vs Risk analysis.

🚀 AI Market Expansion Recommender  
Automatically identifies the best countries for export expansion by industry.

🌎 Global Trade Opportunity Ranking  
Ranks the Top 20 countries globally for trade expansion potential.

🧠 Executive Insight Summary  
Generates consulting-style strategic insights automatically.

## 🧠 Machine Learning Model

The project uses **Linear Regression forecasting models** to predict:

- Export growth trends
- Industry export potential

The model learns patterns from historical trade data and estimates future export performance.
